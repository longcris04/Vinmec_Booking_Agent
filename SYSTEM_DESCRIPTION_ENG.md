## 2. The ReAct Model (Thought → Action → Observation)

The agent iterates through the following cycle until it produces a `Final Answer`:

```
Thought: reason about what to do next
Action: tool_name(arg1=..., arg2=...)
Observation: <tool result injected by CODE — not generated by the LLM>
... (repeat Thought/Action/Observation) ...
Final Answer: the final response to the user
```

**Action convention:** `Action: tool_name(arg1=..., arg2=...)`

**Reliability mechanisms implemented:**
- **CONTINUE_CUE** (`Thought:`): primes the LLM to continue rather than restart.
- **`_truncate()`**: cuts LLM output at `Observation:`/`User:`/`Question:` to prevent hallucinated dialogue turns.
- **Salvage**: if the output has no `Action:` but contains content → treat it as `Final Answer`.
- **Conversation memory**: `_build_transcript` / `_remember` maintains multi-turn context.
- **`max_steps = 8`**: hard safety limit.

## 6. Technical Constraints

1. **Prevent hallucinated Observations:** The LLM must stop before generating `Observation:`. The agent loop uses `_truncate()` to cut hallucinated content, combined with provider-level `stop` tokens.
2. **Robust parser:** Uses AST (`ast.parse`) instead of regex to parse `tool(arg=value)` — safer across the varied formats LLMs produce.
3. **Provider switching:** Reads `DEFAULT_PROVIDER` from `.env` (`openai` | `google` | `openrouter` | `local`).
4. **Telemetry:** Every LLM call is recorded via `tracker.track_request()` — tokens, latency, cost estimate.
5. **Rate limiting (web):** ≤15 messages/minute, ≥2s between messages per IP; message length ≤500 characters.
6. **Medical guardrail:** Out-of-scope questions are rejected; emergency situations are prioritized immediately (115 + first aid) before any appointment advice.

## 7. Models & Configuration

| Provider | Model | When to use |
| :--- | :--- | :--- |
| `google` | `gemini-2.5-flash` | Primary — high quality, free quota available |
| `openrouter` | `google/gemini-3.1-flash-lite` (default) | Flexible — change model via `OPENROUTER_MODEL` |
| `openai` | `gpt-4o` | Backup, quality comparison |
| `local` | `llama3.1:8b` via Ollama | Offline / no API cost |

**`.env` configuration:**
```env
DEFAULT_PROVIDER=openrouter        # openai | google | openrouter | local
GEMINI_API_KEY=...
OPENAI_API_KEY=...
OPENROUTER_API_KEY=...
OPENROUTER_MODEL=google/gemini-3.1-flash-lite
OLLAMA_MODEL=llama3.1:8b           # used when DEFAULT_PROVIDER=local
OLLAMA_BASE_URL=http://localhost:11434
```

**Running the local LLM (Ollama):**
```bash
ollama pull llama3.1:8b
# Set DEFAULT_PROVIDER=local in .env
python main.py
```
